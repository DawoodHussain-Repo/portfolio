# iOS 26 NeoGlass Portfolio — Dawood Hussain

A futuristic portfolio website featuring the iOS 26 NeoGlass design aesthetic - combining glassmorphism, neon gradients, and fluid motion for a cutting-edge user experience.

## 🎨 Design Philosophy

**Theme:** iOS 26 — NeoGlass Portfolio  
**Core Aesthetic:** Futuristic minimalism, glassmorphism, soft neon gradients, fluid motion, and adaptive depth layers  
**Target Feel:** "Apple 2030 Vision" — clean, AI-driven, holographic style with depth and life in every interaction

## ✨ Key Features

- **Glassmorphism UI:** Frosted glass cards with backdrop blur and subtle transparency
- **Neon Gradient Accents:** Cyan-to-purple gradients with glow effects
- **Fluid Animations:** GSAP and Framer Motion for smooth, spring-like interactions
- **Parallax Backgrounds:** Dynamic gradient backgrounds that respond to mouse movement
- **3D Hover Effects:** Cards that tilt and elevate on interaction
- **Responsive Design:** Mobile-first with iOS-style dock navigation
- **GitHub Integration:** Live project showcase from GitHub API

## 🛠️ Tech Stack

- **Frontend:** React 18 + TypeScript
- **Styling:** Tailwind CSS with custom NeoGlass utilities
- **Animations:** Framer Motion + GSAP
- **Build Tool:** Vite
- **UI Components:** Radix UI + shadcn/ui
- **Icons:** Lucide React

## 🎯 Sections

1. **Hero Section** - Floating 3D card with animated name and neon glow
2. **About Me** - Personal introduction with skill highlights
3. **Projects Showcase** - Interactive grid of GitHub projects with modal details
4. **Tech Stack** - Categorized technology display with smooth transitions
5. **Contact/Footer** - Glass contact cards with social links

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production

```bash
# Create optimized production build
npm run build

# Preview production build
npm run preview
```

## 📱 Responsive Design

- **Desktop:** Fixed top navigation with icon-only buttons
- **Mobile:** Bottom dock navigation (iOS-style)
- **Tablet:** Adaptive layouts with optimized spacing

## 🎭 Animation Features

- **Float Animation:** Subtle vertical movement on hero card
- **Parallax Gradient:** Background responds to mouse position
- **Spring Transitions:** Smooth, physics-based animations
- **Hover Effects:** 3D tilt and elevation on cards
- **Scroll Triggers:** Elements animate into view on scroll

## 🔧 Custom CSS Utilities

```css
.neoglass-card - Frosted glass card component
.neoglass-nav - Navigation glass effect
.neoglass-button - Neon gradient button
.frosted-pill - Small glass pill component
.parallax-gradient - Animated gradient background
.text-neon-glow - Neon text glow effect
```
