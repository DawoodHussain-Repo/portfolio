## 📐 Design Specifications

### Project Card Dimensions
- **Border Radius**: 16px (rounded-2xl)
- **Padding**: 24px (p-6)
- **Accent Bar**: 4px → 6px on hover
- **Icon Circle**: 48px diameter
- **Gradient**: from-slate-900 to-slate-800

### Colors Used
- **Background**: `from-slate-900 to-slate-800`
- **Accent Bar**: `from-primary via-primary/50 to-transparent`
- **Icon Circle**: `bg-primary/20` → `bg-primary/30` on hover
- **Tags**: `bg-primary/10 text-primary border-primary/20`
- **Hover Overlay**: `from-primary/20 via-primary/10 to-transparent`

### Animations
- **Transition Duration**: 300ms - 500ms
- **Hover Scale**: Icon 1.1x
- **Opacity Transitions**: 0 → 1 on hover
- **Gradient Shift**: 8s infinite loop
